Projecting Iowa Liquor Sales

The goal of this task was to use data on sales from liquor distributors to liquor stores in Iowa to determine where to open a new store. Each record in the dataset was a different sale, detailing the quantity, prices, what was sold, and when. Using a linear regression to estimate sales in the duration of 2016 based on data for the entire year in 2015 and the first quarter of 2016, I was able to make recommendations for counties in which a store would be most profitable.

Indeed Data Science Salary Predictions

This project required that I obtain my own data set on salaries for data science-related positions to build a model that predicts whether a posted job will fall above or below the median data science salary. This presented a challenge in scraping my own data, determining how much was necessary to build an effective model, and then deciding what features would be necessary to predict that a salary is higher or lower paying.

  • Github: Jupyter Notebook

Bayesian Analysis of Global Terrorism

The University of Maryland's START program maintains the Global Terrorism Database, which provides data on attacks dating back to 1970. The first goal of this task was to conduct a Bayesian inference to compare two different populations, and determine if they are statistically different. The second goal required that I impute the number of terror-related bombings that occurred in 1993, a year in which all data has been lost from the dataset.

  • Github: Jupyter Notebook

Evaluating Excitement of Soccer Matches

The goal for this project was to identify metrics that could differentiate soccer matches in terms of excitement. Using data for individual events that occurred in each match, I engineered several features to measure excitement of a match based on how close the match was, goals scored, and the number of individual events occurring in a match. I used an unsupervised machine learning technique to create clusters of similar types of matches, and then used these to predict if matches in the current season are expected to be exciting to watch, or boring.

  • Github: Folder with Notebooks
  • Visualization of Results: Tableau
